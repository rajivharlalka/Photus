## Photus

- Upload photo and get url to share them.

Uses React Js as Frontend, NodeJS as backend and supabase for storing images

to run locally,

```
cd server
touch .env
add SUPABASE_URL,PUBLIC ANONYMOUS KEY and name of bucket for image upload
```
